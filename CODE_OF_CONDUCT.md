# Code of Conduct

## Our Pledge

Vi vill ha ett öppet, inkluderande och respektfullt samarbete för alla.

## Expected Behavior

- Visa respekt i diskussioner och code reviews
- Ge konstruktiv, konkret feedback
- Anta goda intentioner
- Hjälp till att hålla projektet välkomnande

## Unacceptable Behavior

- Trakasserier eller diskriminering
- Personangrepp, hot eller förolämpningar
- Delning av privat information utan samtycke

## Enforcement

Maintainers kan moderera diskussioner och vid behov ta bort innehåll eller begränsa åtkomst.

## Reporting

Rapportera incidenter privat till projektets maintainers eller via säkra kanaler.
