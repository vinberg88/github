# Support

## Frågor och hjälp

- För buggar/funktioner: använd `Issues` (templates finns aktiverade)
- För säkerhetsärenden: följ `SECURITY.md` (privat rapportering)

## Drift och felsökning (snabbt)

1. Kör `npm run lint`
2. Kör `npm run build`
3. Verifiera miljövariabler i `.env`
4. Kontrollera workflows under `.github/workflows/`

## Kontaktvägar

- Tekniska förbättringar: Pull Request
- Process/Governance: Issue med label `docs` eller `enhancement`
